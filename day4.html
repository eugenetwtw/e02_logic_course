Day 4: Deductive Reasoning and Logical Rules

Understand the basic structure of syllogisms, learn common logical symbols, and master fundamental rules of inference.

Mind map of deductive reasoning and logical rules

Course Objectives

This lesson introduces the basic structure and rules of deductive reasoning. Students will understand the form of syllogisms, learn basic logical symbols, and practice essential inference rules.

After completing this lesson, students will be able to:

  • Understand the features and applications of deductive reasoning
  • Analyze the structure and validity of syllogisms
  • Use basic logical symbols to express propositions and arguments
  • Apply fundamental inference rules
  • Use Venn diagrams to test syllogistic validity

Basics of Deductive Reasoning

Deductive reasoning moves from general principles to specific cases. If the premises are true, the conclusion must be true.

Characteristics of Deductive Reasoning

  • From general to specific: deriving particular cases from general principles
  • Necessity: if the premises are true, the conclusion must be true
  • Certainty: the conclusion has high certainty
  • No new information: the conclusion contains nothing not already implicit in the premises

Example of Deductive Reasoning

All mammals have hearts. (General principle)
All dogs are mammals. (General principle)
Therefore, all dogs have hearts. (Specific case)

In this example the conclusion "All dogs have hearts" follows necessarily from the premises "All mammals have hearts" and "All dogs are mammals." If those premises are true, the conclusion must be true.

Syllogism

A syllogism is a fundamental deductive form consisting of two premises and a conclusion involving three terms: the major, the minor, and the middle.

Structure of a Syllogism

  • Major premise: contains the major and middle terms
  • Minor premise: contains the minor and middle terms
  • Conclusion: contains the minor and major terms

Where:

  • Major term: appears as predicate of the conclusion
  • Middle term: appears in both premises but not in the conclusion
  • Minor term: appears as subject of the conclusion

Example

All men are mortal. (Major premise)
Socrates is a man. (Minor premise)
Therefore, Socrates is mortal. (Conclusion)

In this example:

  • The major term is "mortal"
  • The middle term is "men"
  • The minor term is "Socrates"

Syllogistic Validity

The validity of a syllogism depends on its form rather than its content. A valid form guarantees that if the premises are true, the conclusion must also be true.

Some basic rules for syllogistic validity:

  1. The middle term must be distributed in at least one premise
  2. If a term is distributed in the conclusion it must be distributed in its premise
  3. At least one premise must be affirmative
  4. If one premise is negative, the conclusion must be negative
  5. Two negative premises cannot yield a valid conclusion

Basic Logical Symbols

Logical symbols represent relationships and operations, making arguments concise and precise.

Propositional Symbols

Yesterday we learned these propositional symbols:

  • ∧: conjunction (and)
  • ∨: disjunction (or)
  • ¬: negation (not)
  • →: conditional (if...then)
  • ↔: biconditional (if and only if)

Predicate Logic Symbols

Predicate logic also uses these symbols:

  • ∀: universal quantifier (for all)
  • ∃: existential quantifier (there exists)
  • P(x): predicate meaning "x has property P"

Expressing Propositions with Symbols

For example, "All humans are mortal" can be written as:

∀x(H(x) → M(x))

where H(x) means "x is human" and M(x) means "x is mortal."

"Some students like math" can be written as:

∃x(S(x) ∧ L(x, m))

where S(x) means "x is a student" and L(x, m) means "x likes math."

Basic Inference Rules

Inference rules allow us to derive conclusions from given premises. Here are several important ones:

Modus Ponens

If p → q is true and p is true, then q is true.

If it rains, the ground is wet. (p → q)
It is raining. (p)
Therefore, the ground is wet. (q)

Modus Tollens

If p → q is true and q is false, then p is false.

If it rains, the ground is wet. (p → q)
The ground is not wet. (¬q)
Therefore, it is not raining. (¬p)

Hypothetical Syllogism

If p → q and q → r are true, then p → r is true.

If it rains, the ground is wet. (p → q)
If the ground is wet, shoes get dirty. (q → r)
Therefore, if it rains, shoes get dirty. (p → r)

Disjunctive Syllogism

If p ∨ q is true and ¬p is true, then q is true.

Either it rains or it snows. (p ∨ q)
It is not raining. (¬p)
Therefore, it snows. (q)

Addition

If p is true, then p ∨ q is true (for any q).

It is raining. (p)
Therefore, it rains or it snows. (p ∨ q)

Simplification

If p ∧ q is true, then p is true and q is true.

It is raining and windy. (p ∧ q)
Therefore, it is raining. (p)
Therefore, it is windy. (q)

Conjunction

If p is true and q is true, then p ∧ q is true.

It is raining. (p)
It is windy. (q)
Therefore, it is raining and windy. (p ∧ q)

Validity Proof

A proof of validity demonstrates that an argument is logically sound. Here is a simple example:

Example

Prove the following argument is valid:

1. 如果下雨，則地面濕。（p → q）
2. 如果地面濕，則鞋子會髒。（q → r）
3. 下雨了。（p）
4. 因此，鞋子會髒。（r）

Proof

  1. p → q (premise 1)
  2. q → r (premise 2)
  3. p (premise 3)
  4. q (from 1 and 3 by modus ponens)
  5. r (from 2 and 4 by modus ponens)

Therefore the conclusion r is valid.

Venn Diagrams

Venn diagrams visually represent set relations and can be used to test the validity of syllogisms.

Basic Venn Diagram

Circles represent sets and their overlaps show intersections.

Analyzing a Syllogism with Venn Diagrams

For example, we can use a Venn diagram to analyze this syllogism:

All humans are mortal. (Major premise)
Socrates is human. (Minor premise)
Therefore, Socrates is mortal. (Conclusion)

In the diagram:

  • One circle represents the set of humans
  • Another circle represents the set of mortal things
  • "Socrates" is shown as a point

The major premise places the "human" circle entirely inside the "mortal" circle.

The minor premise locates the point "Socrates" inside the human circle.

Thus Socrates must also lie inside the mortal circle, showing the conclusion is valid.
