第四天：演繹推理與邏輯規則

理解三段論的基本結構，學習基本邏輯符號的使用，掌握基本推理規則。

演繹推理與邏輯規則心智圖

課程目標

本課程旨在介紹演繹推理的基本結構和規則，幫助學生理解三段論的形式，學習基本邏輯符號的使用，以及掌握基本推理規則。

完成本課程後，學生將能夠：

  • 理解演繹推理的特點和應用
  • 分析三段論的結構和有效性
  • 使用基本邏輯符號表達命題和論證
  • 應用基本推理規則進行有效推理
  • 使用文氏圖分析三段論的有效性

演繹推理基礎

演繹推理是從一般原則到特殊情況的推理過程。在演繹推理中，如果前提為真，則結論必然為真。

演繹推理的特點

  • 從一般到特殊：從普遍原則推導出特殊情況
  • 必然性：如果前提為真，則結論必然為真
  • 確定性：結論的確定性高
  • 不增加新信息：結論中的信息已隱含在前提中

演繹推理的例子

所有哺乳動物都有心臟。（一般原則）
所有狗都是哺乳動物。（一般原則）
因此，所有狗都有心臟。（特殊情況）

在這個例子中，結論「所有狗都有心臟」是從前提「所有哺乳動物都有心臟」和「所有狗都是哺乳動物」必然推導出來的。如果這兩個前提為真，則結論必然為真。

三段論

三段論是一種基本的演繹推理形式，由兩個前提和一個結論組成，涉及三個術語：大詞、中詞和小詞。

三段論的結構

  • 大前提：包含大詞和中詞的命題
  • 小前提：包含小詞和中詞的命題
  • 結論：包含小詞和大詞的命題

其中：

  • 大詞：在結論中作謂詞的術語
  • 中詞：在兩個前提中出現但在結論中不出現的術語
  • 小詞：在結論中作主詞的術語

三段論的例子

所有人都會死。（大前提：中詞「人」和大詞「會死」）
蘇格拉底是人。（小前提：小詞「蘇格拉底」和中詞「人」）
因此，蘇格拉底會死。（結論：小詞「蘇格拉底」和大詞「會死」）

在這個例子中：

  • 大詞是「會死」
  • 中詞是「人」
  • 小詞是「蘇格拉底」

三段論的有效性

三段論的有效性取決於其形式，而不是內容。有效的三段論形式保證如果前提為真，則結論必然為真。

三段論有效性的一些基本規則：

  1. 中詞必須至少在一個前提中周延（指涉該類別的所有成員）
  2. 如果一個術語在結論中周延，它也必須在相應的前提中周延
  3. 至少有一個前提必須是肯定命題
  4. 如果有一個前提是否定命題，則結論必須是否定命題
  5. 兩個否定前提不能導出有效結論

基本邏輯符號

邏輯符號是用來表示邏輯關係和操作的符號，它們使邏輯表達更加簡潔和精確。

命題邏輯符號

我們在昨天的課程中已經學習了以下命題邏輯符號：

  • ∧：連言詞，表示「且」
  • ∨：選言詞，表示「或」
  • ¬：否定詞，表示「非」
  • →：條件詞，表示「如果...那麼...」
  • ↔：雙條件詞，表示「當且僅當」

謂詞邏輯符號

除了命題邏輯符號外，謂詞邏輯還使用以下符號：

  • ∀：全稱量詞，表示「對所有...」
  • ∃：存在量詞，表示「存在...」
  • P(x)：謂詞，表示「x 具有性質 P」

使用邏輯符號表達命題

例如，「所有人都會死」可以表示為：

∀x(H(x) → M(x))

其中 H(x) 表示「x 是人」，M(x) 表示「x 會死」。

「有些學生喜歡數學」可以表示為：

∃x(S(x) ∧ L(x, m))

其中 S(x) 表示「x 是學生」，L(x, m) 表示「x 喜歡數學」。

基本推理規則

推理規則是用來從已知前提推導出結論的規則。以下是一些基本的推理規則：

肯定前件（Modus Ponens）

如果 p → q 為真，且 p 為真，則 q 為真。

如果下雨，則地面濕。（p → q）
下雨了。（p）
因此，地面濕。（q）

否定後件（Modus Tollens）

如果 p → q 為真，且 q 為假，則 p 為假。

如果下雨，則地面濕。（p → q）
地面不濕。（¬q）
因此，沒有下雨。（¬p）

假言三段論（Hypothetical Syllogism）

如果 p → q 為真，且 q → r 為真，則 p → r 為真。

如果下雨，則地面濕。（p → q）
如果地面濕，則鞋子會髒。（q → r）
因此，如果下雨，則鞋子會髒。（p → r）

選言三段論（Disjunctive Syllogism）

如果 p ∨ q 為真，且 ¬p 為真，則 q 為真。

要麼下雨，要麼下雪。（p ∨ q）
沒有下雨。（¬p）
因此，下雪了。（q）

附加（Addition）

如果 p 為真，則 p ∨ q 為真（對任意 q）。

下雨了。（p）
因此，要麼下雨，要麼下雪。（p ∨ q）

化簡（Simplification）

如果 p ∧ q 為真，則 p 為真且 q 為真。

下雨且刮風。（p ∧ q）
因此，下雨了。（p）
因此，刮風了。（q）

合取（Conjunction）

如果 p 為真且 q 為真，則 p ∧ q 為真。

下雨了。（p）
刮風了。（q）
因此，下雨且刮風。（p ∧ q）

有效性證明

有效性證明是一種用來證明論證有效性的方法。以下是一個簡單的有效性證明例子：

例子

證明以下論證的有效性：

1. 如果下雨，則地面濕。（p → q）
2. 如果地面濕，則鞋子會髒。（q → r）
3. 下雨了。（p）
4. 因此，鞋子會髒。（r）

證明

  1. p → q（前提1）
  2. q → r（前提2）
  3. p（前提3）
  4. q（由1和3，使用肯定前件規則）
  5. r（由2和4，使用肯定前件規則）

因此，結論 r 是有效的。

文氏圖

文氏圖是一種用來表示集合關係的圖形，它可以用來分析三段論的有效性。

基本文氏圖

文氏圖使用圓圈表示集合，圓圈的重疊部分表示集合的交集。

使用文氏圖分析三段論

例如，我們可以使用文氏圖分析以下三段論：

所有人都會死。（大前提）
蘇格拉底是人。（小前提）
因此，蘇格拉底會死。（結論）

在文氏圖中：

  • 一個圓圈表示「人」的集合
  • 另一個圓圈表示「會死的事物」的集合
  • 「蘇格拉底」表示為一個點

大前提「所有人都會死」表示「人」的圓圈完全包含在「會死的事物」的圓圈內。

小前提「蘇格拉底是人」表示「蘇格拉底」這個點位於「人」的圓圈內。

從文氏圖中，我們可以看到「蘇格拉底」這個點必然也位於「會死的事物」的圓圈內，這就是結論「蘇格拉底會死」。

  • 演繹推理的特點和應用
  • 三段論的結構和有效性
  • 基本邏輯符號的使用
  • 基本推理規則，如肯定前件、否定後件、假言三段論等
