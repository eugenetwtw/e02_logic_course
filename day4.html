第四天：演繹推理 (Inference) (Deductive Reasoning)與邏輯規則

理解三段論 (Syllogism)的基本結構，學習基本邏輯符號的使用，掌握基本推理 (Inference)規則。

演繹推理 (Inference) (Deductive Reasoning)與邏輯規則心智圖

課程目標

本課程旨在介紹演繹推理 (Inference) (Deductive Reasoning)的基本結構和規則，幫助學生理解三段論 (Syllogism)的形式，學習基本邏輯符號的使用，以及掌握基本推理 (Inference)規則。

完成本課程後，學生將能夠：

  • 理解演繹推理 (Inference) (Deductive Reasoning)的特點和應用
  • 分析三段論 (Syllogism)的結構和有效性
  • 使用基本邏輯符號表達命題 (Proposition)和論證 (Argument)
  • 應用基本推理 (Inference)規則進行有效推理 (Inference)
  • 使用文氏圖分析三段論 (Syllogism)的有效性

演繹推理 (Inference) (Deductive Reasoning)基礎

演繹推理 (Inference) (Deductive Reasoning)是從一般原則到特殊情況的推理 (Inference)過程。在演繹推理 (Inference) (Deductive Reasoning)中，如果前提 (Premise)為真，則結論 (Conclusion)必然為真。

演繹推理 (Inference) (Deductive Reasoning)的特點

  • 從一般到特殊：從普遍原則推導出特殊情況
  • 必然性：如果前提 (Premise)為真，則結論 (Conclusion)必然為真
  • 確定性：結論 (Conclusion)的確定性高
  • 不增加新信息：結論 (Conclusion)中的信息已隱含在前提 (Premise)中

演繹推理 (Inference) (Deductive Reasoning)的例子

所有哺乳動物都有心臟。（一般原則）
所有狗都是哺乳動物。（一般原則）
因此，所有狗都有心臟。（特殊情況）

在這個例子中，結論 (Conclusion)「所有狗都有心臟」是從前提 (Premise)「所有哺乳動物都有心臟」和「所有狗都是哺乳動物」必然推導出來的。如果這兩個前提 (Premise)為真，則結論 (Conclusion)必然為真。

三段論 (Syllogism)

三段論 (Syllogism)是一種基本的演繹推理 (Inference) (Deductive Reasoning)形式，由兩個前提 (Premise)和一個結論 (Conclusion)組成，涉及三個術語：大詞、中詞和小詞。

三段論 (Syllogism)的結構

  • 大前提 (Premise)：包含大詞和中詞的命題 (Proposition)
  • 小前提 (Premise)：包含小詞和中詞的命題 (Proposition)
  • 結論 (Conclusion)：包含小詞和大詞的命題 (Proposition)

其中：

  • 大詞：在結論 (Conclusion)中作謂詞的術語
  • 中詞：在兩個前提 (Premise)中出現但在結論 (Conclusion)中不出現的術語
  • 小詞：在結論 (Conclusion)中作主詞的術語

三段論 (Syllogism)的例子

所有人都會死。（大前提 (Premise)：中詞「人」和大詞「會死」）
蘇格拉底是人。（小前提 (Premise)：小詞「蘇格拉底」和中詞「人」）
因此，蘇格拉底會死。（結論 (Conclusion)：小詞「蘇格拉底」和大詞「會死」）

在這個例子中：

  • 大詞是「會死」
  • 中詞是「人」
  • 小詞是「蘇格拉底」

三段論 (Syllogism)的有效性

三段論 (Syllogism)的有效性取決於其形式，而不是內容。有效的三段論 (Syllogism)形式保證如果前提 (Premise)為真，則結論 (Conclusion)必然為真。

三段論 (Syllogism)有效性的一些基本規則：

  1. 中詞必須至少在一個前提 (Premise)中周延（指涉該類別的所有成員）
  2. 如果一個術語在結論 (Conclusion)中周延，它也必須在相應的前提 (Premise)中周延
  3. 至少有一個前提 (Premise)必須是肯定命題 (Proposition)
  4. 如果有一個前提 (Premise)是否定 (Negation)命題 (Proposition)，則結論 (Conclusion)必須是否定 (Negation)命題 (Proposition)
  5. 兩個否定 (Negation)前提 (Premise)不能導出有效結論 (Conclusion)

基本邏輯符號

邏輯符號是用來表示邏輯關係和操作的符號，它們使邏輯表達更加簡潔和精確。

命題 (Proposition)邏輯符號

我們在昨天的課程中已經學習了以下命題 (Proposition)邏輯符號：

  • ∧：連言詞，表示「且」
  • ∨：選言 (Disjunction)詞，表示「或」
  • ¬：否定 (Negation)詞，表示「非」
  • →：條件詞，表示「如果...那麼...」
  • ↔：雙條件詞，表示「當且僅當」

謂詞邏輯符號

除了命題 (Proposition)邏輯符號外，謂詞邏輯還使用以下符號：

  • ∀：全稱量詞，表示「對所有...」
  • ∃：存在量詞，表示「存在...」
  • P(x)：謂詞，表示「x 具有性質 P」

使用邏輯符號表達命題 (Proposition)

例如，「所有人都會死」可以表示為：

∀x(H(x) → M(x))

其中 H(x) 表示「x 是人」，M(x) 表示「x 會死」。

「有些學生喜歡數學」可以表示為：

∃x(S(x) ∧ L(x, m))

其中 S(x) 表示「x 是學生」，L(x, m) 表示「x 喜歡數學」。

基本推理 (Inference)規則

推理 (Inference)規則是用來從已知前提 (Premise)推導出結論 (Conclusion)的規則。以下是一些基本的推理 (Inference)規則：

肯定前件（Modus Ponens）

如果 p → q 為真，且 p 為真，則 q 為真。

如果下雨，則地面濕。（p → q）
下雨了。（p）
因此，地面濕。（q）

否定 (Negation)後件（Modus Tollens）

如果 p → q 為真，且 q 為假，則 p 為假。

如果下雨，則地面濕。（p → q）
地面不濕。（¬q）
因此，沒有下雨。（¬p）

假言三段論 (Syllogism) (Hypothetical Syllogism)（Hypothetical Syllogism）

如果 p → q 為真，且 q → r 為真，則 p → r 為真。

如果下雨，則地面濕。（p → q）
如果地面濕，則鞋子會髒。（q → r）
因此，如果下雨，則鞋子會髒。（p → r）

選言 (Disjunction)三段論 (Syllogism) (Disjunctive Syllogism)（Disjunctive Syllogism）

如果 p ∨ q 為真，且 ¬p 為真，則 q 為真。

要麼下雨，要麼下雪。（p ∨ q）
沒有下雨。（¬p）
因此，下雪了。（q）

附加（Addition）

如果 p 為真，則 p ∨ q 為真（對任意 q）。

下雨了。（p）
因此，要麼下雨，要麼下雪。（p ∨ q）

化簡（Simplification）

如果 p ∧ q 為真，則 p 為真且 q 為真。

下雨且刮風。（p ∧ q）
因此，下雨了。（p）
因此，刮風了。（q）

合取（Conjunction）

如果 p 為真且 q 為真，則 p ∧ q 為真。

下雨了。（p）
刮風了。（q）
因此，下雨且刮風。（p ∧ q）

有效性證明

有效性證明是一種用來證明論證 (Argument)有效性的方法。以下是一個簡單的有效性證明例子：

例子

證明以下論證 (Argument)的有效性：

1. 如果下雨，則地面濕。（p → q）
2. 如果地面濕，則鞋子會髒。（q → r）
3. 下雨了。（p）
4. 因此，鞋子會髒。（r）

證明

  1. p → q（前提 (Premise)1）
  2. q → r（前提 (Premise)2）
  3. p（前提 (Premise)3）
  4. q（由1和3，使用肯定前件規則）
  5. r（由2和4，使用肯定前件規則）

因此，結論 (Conclusion) r 是有效的。

文氏圖

文氏圖是一種用來表示集合關係的圖形，它可以用來分析三段論 (Syllogism)的有效性。

基本文氏圖

文氏圖使用圓圈表示集合，圓圈的重疊部分表示集合的交集。

使用文氏圖分析三段論 (Syllogism)

例如，我們可以使用文氏圖分析以下三段論 (Syllogism)：

所有人都會死。（大前提 (Premise)）
蘇格拉底是人。（小前提 (Premise)）
因此，蘇格拉底會死。（結論 (Conclusion)）

在文氏圖中：

  • 一個圓圈表示「人」的集合
  • 另一個圓圈表示「會死的事物」的集合
  • 「蘇格拉底」表示為一個點

大前提 (Premise)「所有人都會死」表示「人」的圓圈完全包含在「會死的事物」的圓圈內。

小前提 (Premise)「蘇格拉底是人」表示「蘇格拉底」這個點位於「人」的圓圈內。

從文氏圖中，我們可以看到「蘇格拉底」這個點必然也位於「會死的事物」的圓圈內，這就是結論 (Conclusion)「蘇格拉底會死」。
