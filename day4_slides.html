第四天：演繹推理 (Inference) (Deductive Reasoning)與邏輯規則

課程目標

  • 理解演繹推理 (Inference) (Deductive Reasoning)的特點和應用
  • 分析三段論 (Syllogism)的結構和有效性
  • 使用基本邏輯符號表達命題 (Proposition)和論證 (Argument)
  • 應用基本推理 (Inference)規則進行有效推理 (Inference)
  • 使用文氏圖分析三段論 (Syllogism)的有效性

演繹推理 (Inference) (Deductive Reasoning)基礎

演繹推理 (Inference) (Deductive Reasoning)是從一般原則到特殊情況的推理 (Inference)過程。

特點：

  • 從一般到特殊
  • 必然性：如果前提 (Premise)為真，則結論 (Conclusion)必然為真
  • 確定性：結論 (Conclusion)的確定性高
  • 不增加新信息：結論 (Conclusion)中的信息已隱含在前提 (Premise)中

演繹推理 (Inference) (Deductive Reasoning)的例子

所有哺乳動物都有心臟。（一般原則）
所有狗都是哺乳動物。（一般原則）
因此，所有狗都有心臟。（特殊情況）

三段論 (Syllogism)的結構

  • 大前提 (Premise)：包含大詞和中詞的命題 (Proposition)
  • 小前提 (Premise)：包含小詞和中詞的命題 (Proposition)
  • 結論 (Conclusion)：包含小詞和大詞的命題 (Proposition)

其中：

  • 大詞：在結論 (Conclusion)中作謂詞的術語
  • 中詞：在兩個前提 (Premise)中出現但在結論 (Conclusion)中不出現的術語
  • 小詞：在結論 (Conclusion)中作主詞的術語

三段論 (Syllogism)的例子

所有人都會死。（大前提 (Premise)：中詞「人」和大詞「會死」）
蘇格拉底是人。（小前提 (Premise)：小詞「蘇格拉底」和中詞「人」）
因此，蘇格拉底會死。（結論 (Conclusion)：小詞「蘇格拉底」和大詞「會死」）

在這個例子中：

  • 大詞是「會死」
  • 中詞是「人」
  • 小詞是「蘇格拉底」

三段論 (Syllogism)的有效性

三段論 (Syllogism)的有效性取決於其形式，而不是內容。

三段論 (Syllogism)有效性的一些基本規則：

  1. 中詞必須至少在一個前提 (Premise)中周延
  2. 如果一個術語在結論 (Conclusion)中周延，它也必須在相應的前提 (Premise)中周延
  3. 至少有一個前提 (Premise)必須是肯定命題 (Proposition)
  4. 如果有一個前提 (Premise)是否定 (Negation)命題 (Proposition)，則結論 (Conclusion)必須是否定 (Negation)命題 (Proposition)
  5. 兩個否定 (Negation)前提 (Premise)不能導出有效結論 (Conclusion)

基本邏輯符號

命題 (Proposition)邏輯符號：

  • ∧：連言詞，表示「且」
  • ∨：選言 (Disjunction)詞，表示「或」
  • ¬：否定 (Negation)詞，表示「非」
  • →：條件詞，表示「如果...那麼...」
  • ↔：雙條件詞，表示「當且僅當」

謂詞邏輯符號

謂詞邏輯符號：

  • ∀：全稱量詞，表示「對所有...」
  • ∃：存在量詞，表示「存在...」
  • P(x)：謂詞，表示「x 具有性質 P」

使用邏輯符號表達命題 (Proposition)

「所有人都會死」可以表示為：

∀x(H(x) → M(x))

其中 H(x) 表示「x 是人」，M(x) 表示「x 會死」。

「有些學生喜歡數學」可以表示為：

∃x(S(x) ∧ L(x, m))

其中 S(x) 表示「x 是學生」，L(x, m) 表示「x 喜歡數學」。

肯定前件（Modus Ponens）

如果 p → q 為真，且 p 為真，則 q 為真。

如果下雨，則地面濕。（p → q）
下雨了。（p）
因此，地面濕。（q）

否定 (Negation)後件（Modus Tollens）

如果 p → q 為真，且 q 為假，則 p 為假。

如果下雨，則地面濕。（p → q）
地面不濕。（¬q）
因此，沒有下雨。（¬p）

假言三段論 (Syllogism) (Hypothetical Syllogism)（Hypothetical Syllogism）

如果 p → q 為真，且 q → r 為真，則 p → r 為真。

如果下雨，則地面濕。（p → q）
如果地面濕，則鞋子會髒。（q → r）
因此，如果下雨，則鞋子會髒。（p → r）

選言 (Disjunction)三段論 (Syllogism) (Disjunctive Syllogism)（Disjunctive Syllogism）

如果 p ∨ q 為真，且 ¬p 為真，則 q 為真。

要麼下雨，要麼下雪。（p ∨ q）
沒有下雨。（¬p）
因此，下雪了。（q）

有效性證明

證明以下論證 (Argument)的有效性：

1. 如果下雨，則地面濕。（p → q）
2. 如果地面濕，則鞋子會髒。（q → r）
3. 下雨了。（p）
4. 因此，鞋子會髒。（r）

證明：

  1. p → q（前提 (Premise)1）
  2. q → r（前提 (Premise)2）
  3. p（前提 (Premise)3）
  4. q（由1和3，使用肯定前件規則）
  5. r（由2和4，使用肯定前件規則）

文氏圖

文氏圖是一種用來表示集合關係的圖形，它可以用來分析三段論 (Syllogism)的有效性。

例如，我們可以使用文氏圖分析以下三段論 (Syllogism)：

所有人都會死。（大前提 (Premise)）
蘇格拉底是人。（小前提 (Premise)）
因此，蘇格拉底會死。（結論 (Conclusion)）
