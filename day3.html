Day 3: Logical Connectives and Compound Statements

Learn the basic connectives used to build compound statements and understand the logic of conditional and biconditional sentences.

Mind map of logical connectives and compound statements

Course Objectives

This lesson introduces the core ideas behind logical connectives and how to use them to create compound statements. Students will also learn how to analyze the truth conditions of these statements.

After completing this lesson, students will be able to:

  • Understand the meaning and use of the five basic logical connectives
  • Build compound statements using logical connectives
  • Analyze the structure of conditional and biconditional statements
  • Use truth tables to examine the truth conditions of compound statements

Basics of Logical Connectives

Logical connectives link simple statements together to form compound statements. In logic there are five basic connectives:

  • Conjunction (∧): represents "and"
  • Disjunction (∨): represents "or"
  • Negation (¬): represents "not"
  • Conditional (→): represents "if...then"
  • Biconditional (↔): represents "if and only if"

These connectives can create many kinds of compound statements, each with its own truth conditions.

Conjunction (AND)

The conjunction expresses "and." It joins two statements to form a compound statement.

Symbol and Form

The symbol for conjunction is "∧". If p and q are statements, then "p ∧ q" means "p and q".

Truth Conditions

"p ∧ q" is true when both p and q are true. If either is false, "p ∧ q" is false.

Truth table for conjunction:

p q p ∧ q
True True True
True False False
False True False
False False False

Example

Consider the following statements:

  • p: Taipei is the capital of Taiwan.
  • q: Taiwan is an island.

Then "p ∧ q" means "Taipei is the capital of Taiwan and Taiwan is an island." Since both p and q are true, "p ∧ q" is true.

Disjunction (OR)

A disjunction expresses "or." It connects two statements to create a compound statement.

Symbol and Form

The symbol for disjunction is "∨". If p and q are statements, then "p ∨ q" means "p or q".

Truth Conditions

"p ∨ q" is true if either p or q is true (or both). It is false only when both p and q are false.

Truth table for disjunction:

p q p ∨ q
True True True
True False True
False True True
False False False

Example

Consider the following statements:

  • p: Today is Saturday.
  • q: Today is Sunday.

Then "p ∨ q" means "Today is Saturday or today is Sunday." If today is Saturday, p is true and q is false, so "p ∨ q" is true. If today is Sunday, p is false and q is true, so it is also true. If today is neither, both p and q are false and the disjunction is false.

Negation (NOT)

Negation expresses "not." It operates on a statement to form its denial.

Symbol and Form

The symbol for negation is "¬". If p is a statement, then "¬p" means "not p".

Truth Conditions

The truth of "¬p" is opposite to the truth of p. If p is true, "¬p" is false, and vice versa.

Truth table for negation:

p ¬p
True False
False True

Example

Consider the following statement:

  • p: The Earth is round.

Then "¬p" states "The Earth is not round." Since p is true, "¬p" is false.

Conditional Statement

A conditional expresses the relationship "if ... then ..." by connecting two statements to form a compound statement.

Symbol and Form

The symbol for a conditional is "→". If p and q are statements, then "p → q" means "if p, then q".

Truth Conditions

The conditional "p → q" is false only when p is true and q is false. In all other cases it is true.

Truth table for the conditional:

p q p → q
True True True
True False False
False True True
False False True

Example

Consider the following statements:

  • p: It is raining.
  • q: The ground is wet.

Then "p → q" means "If it rains, the ground gets wet." This conditional is false only if it rains but the ground is not wet. In all other cases it is true.

Biconditional Statement (IF AND ONLY IF)

A biconditional expresses "if and only if." It joins two statements so that each implies the other.

Symbol and Form

The symbol for a biconditional is "↔". If p and q are statements, then "p ↔ q" means "p if and only if q".

Truth Conditions

The biconditional "p ↔ q" is true when p and q have the same truth value—that is, both true or both false.

Truth table for the biconditional:

p q p ↔ q
True True True
True False False
False True False
False False True

Example

Consider the following statements:

  • p: A number is even.
  • q: The number is divisible by 2.

Then "p ↔ q" means "A number is even if and only if it is divisible by 2." This biconditional is true because these two statements are equivalent.

Truth Tables

A truth table is a tool for analyzing the truth conditions of compound statements. It lists all possible combinations of truth values and shows the result for the compound statement in each case.

Steps to Build a Truth Table

  1. Identify the simple statements (atomic propositions)
  2. List all possible truth-value combinations
  3. Use the truth conditions of the connectives to compute the truth value of the compound statement

Example

Let's build a truth table for "(p ∧ q) → r".

First identify the simple statements: p, q, and r.

Then list all possible truth-value combinations:

p q r p ∧ q (p ∧ q) → r
True True True True True
True True False True False
True False True False True
True False False False True
False True True False True
False True False False True
False False True False True
False False False False True

From the truth table we can see that "(p ∧ q) → r" is true in most cases, and false only when p and q are true but r is false.

Course Summary

Today we covered:

  • The five basic logical connectives: conjunction (∧), disjunction (∨), negation (¬), conditional (→), and biconditional (↔)
  • The meaning and truth conditions of each connective
  • How to build compound statements using these connectives
  • How to analyze compound statements with truth tables

These ideas form the basis for understanding more complex logical structures and inference rules.
