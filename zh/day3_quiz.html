Review Points

  1. Conjunction (And, ∧): p ∧ q is true only when p is true and q is true
  2. Disjunction (Or, ∨): p ∨ q is true when p is true or q is true (or both are true)
  3. Negation (Not, ¬): The truth value of ¬p is the opposite of the truth value of p
  4. Conditional (If...then, →): p → q is false only when p is true and q is false
  5. Biconditional (If and only if, ↔): p ↔ q is true when p and q have the same truth value
  6. Truth tables are tools for analyzing the truth conditions of compound statements by listing all possible truth value combinations
